Công việc chính của bài này là kiểm tra xem một số có thuộc dãy hay không, việc tìm dãy con dài nhất còn lại khá đơn giản. Có nhiều cách để kiểm tra một số có nằm trong dãy hay không.

Cách thứ nhất, ta sinh dãy . Giới hạn số trong bài là nên có khoảng số cần sinh. Sau khi sinh ta có thể kiểm tra bằng tìm kiếm nhị phân hoặc cấu trúc Set/Bitset của STL.

Cách thứ 2, ta kiểm tra trực tiếp. Dễ thấy là tổng các số nguyên từ 1 đến nên:

Để kiểm tra xem có nằm trong hay không, ta tìm ( là hàm làm tròn dưới), sau đó, thuộc nếu .

#include <iostream>
#include <vector>
#include <cmath>
using namespace std;

bool test(int a) {
    int k = sqrt(2*a);
    return k*(k+1) / 2 == a;
}

int main() {
    ios::sync_with_stdio(false); cin.tie(0);
    int n; cin >> n;
    vector<int> a(n);
    for (int &a: a) cin >> a;

    int l = 0, max_l = 0;
    for (int i=0; i<n; i++) {
        if (test(a[i])) {
            if (i>0 && a[i]>=a[i-1]) l += 1;
            else l = 1;
            max_l = max(max_l, l);
        } else l = 0;
    }

    cout << max_l;

    return 0;
}
