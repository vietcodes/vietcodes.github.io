Ta tính mảng tổng là tổng từ đến , . Sau đó sắp xếp lại mảng tăng dần (có lưu lại chỉ số ban đầu của phần tử).

Lần lượt duyệt các phần tử trong mảng đã sắp lại. Ở , vì đề yêu cầu tổng của dãy con phải không nhỏ hơn nên ta cần tìm lớn nhất thỏa . Khi đó dãy con dài nhất tại là , là mảng chỉ số ban đầu của các phần tử trong dãy .

Để tìm , có thể sử dụng tìm kiếm nhị phân, cho độ phức tạp của lời giải là . Tuy nhiên, do ta duyệt tăng dần nên cũng sẽ tăng dần theo Từ đó có cách làm như bên dưới, tại mỗi bước ta tăng trong khi điều kiện còn thỏa.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

typedef pair<int,int> pii;

int main() {
    ios::sync_with_stdio(false); cin.tie(0);

    int n, p; cin >> n >> p;
    vector<pii> s(n+1);
    s[0] = {0, 0};
    for (int i=1; i<=n; i++) {
        int a; cin >> a;
        s[i] = {s[i-1].first + a,i};
    }

    sort(s.begin(), s.end());
    int res = 0;
    int min_index = n;
    for (int i=0, j=-1; i<=n; i++) {
        while (j<n && s[i].first-s[j+1].first >= p) {
            min_index = min(min_index, s[++j].second);
        }
        if (j>=0) {
            res = max(res, s[i].second - min_index);
        }
    }

    if (res == 0) res = -1;
    cout << res;
    return 0;
}
